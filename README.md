# Superstore Sales Analytics Dashboard

## Project Overview
This project analyzes Superstore sales data to identify sales trends, profitability, customer behavior, and regional performance. Data was cleaned using Excel, analyzed using SQL, and visualized through an interactive Power BI dashboard.

## Tools Used
- Excel (Data Cleaning & Preparation)
- MySQL (Data Analysis)
- Power BI (Dashboard Development)

## Key KPIs
- Total Sales
- Total Profit
- Total Orders
- Profit Margin
- Sales by Category
- Sales by Region
- Top Customers

## Analysis Performed
- Regional Sales Analysis
- Category & Sub-Category Performance
- Customer Segment Analysis
- Monthly Sales Trend
- Profitability Analysis

## Dashboard Features
- Interactive Filters
- KPI Cards
- Trend Analysis
- Geographic Insights
- Customer Insights

## Skills Demonstrated
- Data Cleaning
- SQL Query Writing
- Data Modeling
- DAX Measures
- Data Visualization
- Business Intelligence
